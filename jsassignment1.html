Name 10 Programming languages and what website used them.


1. Objective-C - Used for developing iOS and macOS applications like Snapchat and Evernote

2. Go - Used for creating websites with high scalability such as Uber and Dropbox

3. TypeScript - Used for developing web applications like Microsoft Office and Slack

4. Rust - Used for building systems and web technologies such as browser engine Servo

5. Kotlin - Used for developing Android applications for Pinterest and Uber

6. Perl - Used primarily for web development and used for building websites like Craigslist

7. SQL - Used for managing and manipulating databases for websites such as Amazon and Google

8. C++ - Used to develop web browsers such as Google Chrome and Firefox

9. Scala - Used for developing web applications for LinkedIn and The Guardian

10. HTML - Used for creating web pages for virtually every website on the internet
